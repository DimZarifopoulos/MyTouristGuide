MyBeach.GR - Tourist Guide Application

MyBeach.GR is a C#-based Windows Forms application developed as the final project for the "Rapid Application Development" course in the MSc in Informatics. The application serves as a comprehensive tourist guide for Greek destinations, featuring interactive content and user-specific management.
Key Features

The application is designed to meet the following requirements:

- User Management & Database: Support for two user roles (guests and registered users) with a login form. The system integrates a database for managing user accounts and data.

- History Tracking: The application keeps a record of visited sections (forms) for each user, allowing them to distinguish between visited and unvisited pages, with the option to clear history.

- Multimedia Content: The guide includes text in Greek, images, and various multimedia files such as audio, animations, and video.

- Text-to-Speech (TTS): Functionality to read aloud informational text blocks for the user.

- Slideshow: An automated image gallery/slideshow implemented using a timer.

- Data Export: Users can export information regarding specific entries (e.g., restaurant or site details) into an external text file.

- Navigation & Menus: Multiple forms for various tourist sections, each featuring a dedicated navigation menu.

Utilities:

- "About" button displaying creator information.

- "Exit" button for closing the application.

- Integrated "Help" section for user guidance.

- Interactive Events: Implementation of advanced events beyond simple button clicks.

Technologies Used

- Language: C# 

- Framework: Windows Forms Application 

- Development Environment: Visual Studio 

Installation

- Clone this repository to your local machine:
git clone <your-repository-url>

- Open the .sln file using Visual Studio.

- Ensure the database connection string is configured correctly according to your local environment.

- Build and run the project.
